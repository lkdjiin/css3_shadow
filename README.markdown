CSS3 Drop Shadow Generator
============================

A Rails/Coffeescript application to build nice and original CSS3 box shadows.

Usage
--------------------------

See it [online](http://css3-drop-shadows.herokuapp.com/).

Dependencies
--------------------------

  * ruby >= 1.9.2
  * rails >= 3.1
